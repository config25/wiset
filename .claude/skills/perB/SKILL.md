# perB — 이직 준비 · 화학 바이오 · 기술직

이 스킬은 **크롬 확장(브라우저 자동화 Claude)에 그대로 붙여넣을 단일 지침**을 만들어 출력한다.
※ 이 CLI는 브라우저를 직접 클릭하지 못한다 → 출력물을 복사해 크롬 확장 대화창에 붙여 실행한다.

## 할 일 (Claude Code가 수행)
1. `dodo/qa/browser-test-mapping.md`(클릭 매핑 전체 규칙)를 Read 한다.
2. 그 규칙 + 아래 [핵심값] + [페르소나 데이터]를 결합해, `/persona-select` → `/analyzing` 까지
   위저드를 채우는 **자기완결 브라우저 지침 한 덩어리**를 출력한다. (에이전트가 파일을 못 읽는다는 전제로, 필요한 규칙을 지침 안에 인라인 포함)
3. 헷갈리는 값은 반드시 지침에 못박는다: 카드 선택 / 희망 업종(관련직) / 고용형태(시간선택제→시간제) /
   근무지 / **없는 선택섹션은 미입력** / **consulting_log는 입력 안 함(DB 자동연동)** /
   폼은 기본 유저 데이터로 채워져 있으니 각 섹션 삭제 후 재입력.

## [핵심값] 이 페르소나 (추측 금지)
- 페르소나 카드: **02 이직 준비** — `?persona=2` 를 모든 페이지에 유지
- 신입/경력: **경력**
- 희망 업종 `#industry`(드롭다운은 관련직): **화학/식품가공 관련직**  ← 카테고리 "화학 바이오"의 화면값. 이 값을 그대로 선택.
- 희망 직무 `#job`: **기술직**
- target_role(참고): `[화학 바이오 - 기술직]`

## [페르소나 데이터] 위저드 입력 소스
- `user_profile` = 페르소나/희망조건/세부고민, `unstructured_data` = 이력서+자소서+포트폴리오.
