> 감점은 아래 "남은 핵심 항목"의 미완 사유를 반영.
> 기준일: 2026-06-26

## 4) 화면별 진행 현황

### 진단 플로우 (입력 단계)
- **4-1) 페르소나 선택 ( 90% )** — 프론트·선택 저장 정상. AI 분기에만 사용, 잔여=문구/예외처리.
- **4-2) 현 상황 입력 ( 80% )** — 학력·경력·자격·어학·활동·자소서·포트폴리오 즉시저장 DB 연동 완료. 잔여=**AI측 포트폴리오/자소서 조회 연동**, 코드맵 일부.
- **4-3) 경력개발 목표 ( 80% )** — 희망 업종/직무/근무지/고용형태 + 스크랩 타겟공고 조회·저장 연동. 잔여=코드맵 엣지, 성장형(career-goal-growth) 마감.
- **4-4) 세부 고민 ( 80% )** — 입력·저장 연동 완료. 잔여=**1:1 컨설팅 결과 연동(현재 더미 "3건")**.
- **4-5) 입력 데이터 확인(검토) ( 75% )** — 저장 데이터 취합 표시. 잔여=**컨설팅 Q&A 연동**, **AI 입력용 포트폴리오/자소서 노출**.
- **분석 중 / 분석 완료 ( 70% )** — 일괄 저장 + AI 호출 트리거 동작. 잔여=실패/재시도 UX, 진행상태 표시.

### AI 리포트
- **4-6) AI 코칭 ( 55% )** — 저장된 리포트(JSON/TEXT) 렌더 + 배너·칩 연동. 잔여=**실시간 AI 답변 표시(현재 저장본 폴백)**, 생성 파이프라인 안정화.
- **4-7) AI 리포트 - 활동분석 ( 45% )** — 기준정합도 역량 + 강점/보완 TOP3 적재·표시. 잔여=**시장정합도 역량(시드 상태)**, **CFI 실측 연동**, JD 비교 라이브화.
- **4-8) AI 리포트 - 액션 플랜 ( 35% )** — 플래너/추천을 기존 테이블로 선연결·표시. 잔여=**AI 추천활동·교육프로그램·부족역량 키워드 연동**.

### 마이페이지
- **4-9) 마이페이지 대시보드 ( 85% )** — 프로필 헤더·역량추이·액션·진단요약 DB 연동 완료.
- **4-10) 마이페이지 진단 이력 ( 85% )** — 진단/코칭 타임라인 + 만족도·실천수 집계 연동.
- **4-11) 마이페이지 액션 플래너 ( 85% )** — 담기/상태변경/삭제 + 추천 조회 연동.

### 관리자
- **4-12) 관리자 대시보드 ( 75% )** — KPI·퍼널·품질·실시간 지표 실데이터 연동. 잔여=**실제 CPU/GPU·이탈률 측정(현재 목업)**.
- **4-13) 관리자 리포트 관리 ( 85% )** — 목록·필터·페이지네이션·CSV 실데이터 연동.
- **4-14) 관리자 만족도 ( 85% )** — 요약·불만 TOP5·분포 실데이터 연동.
- **4-15) 관리자 AI 품질 ( 75% )** — 지표·가중치 조회/저장 연동. 잔여=**프롬프트 평가·실측 성능지표**.

## 5) 남은 핵심 항목 (감점 사유)

**연동 미완**
- ~~컨설팅 결과 연동~~ → ✅ **완료(2026-06-26)**: `current-situation`/`review` 화면이 `TN_CNSL_REQST_INFO` 실제 Q&A(세션 사용자 기준)를 동적 건수·목록으로 표시. (AI 입력 투입·저품질 필터·PII 마스킹은 별도 트랙)

**API (AI 연동 미완)**
- **입력 데이터 확인** — AI측에서 포트폴리오·자기소개서 조회/전달
- **활동분석** — 시장정합도 역량 / CFI 실측 연동
- **액션 플랜** — AI 추천활동, 교육프로그램 및 부족역량 키워드
- **관리자** — 실제 CPU/GPU 사용률·이탈률 등 실측 지표 (현재 목업/시드)
