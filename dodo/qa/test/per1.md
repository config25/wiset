## 핵심 기술 패턴 (성공 검증됨)
1. **모달 내 항목 삭제** (X버튼은 hover 시에만 보임):
   ```javascript
   const modal = document.querySelector('.wb-modal-overlay');
   modal.querySelectorAll('.wb-rec-x')[0].click();
   ```
   → 이후 확인 다이얼로그의 "확인" 클릭.
2. **페이지의 학력/경력 행 삭제**:
   ```javascript
   document.querySelectorAll('.row').forEach(row => {
     if (row.textContent.includes('삭제할텍스트')) {
       const ch = Array.from(row.children); ch[ch.length-1].click();
     }
   });
   ```
3. **희망 근무지 태그 삭제**:
   ```javascript
   document.querySelectorAll('span.row.items-center.gap-6').forEach(span => {
     if (span.textContent.includes('삭제할도시')) {
       const s = span.querySelectorAll('span'); s[s.length-1].click();
     }
   });
   ```
4. 포트폴리오 URL X버튼은 좌표 클릭으로 제거.
5. **모든 버튼은 find → ref 추출 → ref로 클릭** (좌표 클릭 지양 — 화면 이동 오류 위험).

## 주의사항
- 기존 데이터는 반드시 **전량 삭제 후** 입력.
- "다음 단계"·"정규직" 등 이동/토글 버튼은 반드시 **ref로 클릭**.
- 없는 선택 섹션(예: 데이터에 항목 없음)은 **미입력으로 비움**.
- **컨설팅 이력은 화면 입력 대상 아님** (DB 자동연동). 손대지 말 것.
- "AI 분석 시작하기" 클릭 후 추가 작업 없음 — 분석 로딩 화면이 뜨면 **즉시 정지**.
